## Built With

To Build this application, I used HTML5 and CSS3 for building the structure and styling it, JavaScript and ES6 classes were used to encapsulate calculator logics and DOM manipulation.

- JavaScript
- ES6 Classes
- HTML
- CSS (FlexBox/Grid)
